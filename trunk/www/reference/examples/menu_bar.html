Menus in a Toolbar

Menus are typically used with a toolbar to create a menu bar for use in panels, dialogs or just in the layout as shown here. This menu illustrates several different aspects of using menus in one consolidated example, including menu items, sub menus, toggle menu items, and grouped toggle menu items.
